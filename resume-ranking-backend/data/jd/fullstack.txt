Job Overview
We are seeking a talented Full Stack Developer to join our dynamic engineering team. Develop robust server-side applications using Node.js, Python, Java, or PHP

Design and implement RESTful APIs and GraphQL endpoints

Deploy applications to cloud platforms (AWS, Azure, Google Cloud)

Implement CI/CD pipelines for automated testing and deployment

Monitor application performance and troubleshoot production issues
